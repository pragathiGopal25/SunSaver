Requirements and notices before running the application: 
- The application runs on the Android studio version Ladybug
- In order to use the application you need access to the internet.
- Please note that you cannot type norwegian characters Æ, Ø and Å on your keyboard in the app. However, it is possible to 
write these characters using the keyboard in the emulator.

How to run the application: 
- In order to run the application, do the following steps:
  1. Download the project files from github
  2. Open these files on android studio
  3. Sync gradle
  4. Choose "Resizable (Experimental) API 34" as the device
  5. Run the app!

Libraries used:
- vico - compose
    - This is the library we used to make/design the graph in our app. 
- Room
    - We used the Room database to store the data in our app.

    - We utilised Hilt for dependency injection in our code.
